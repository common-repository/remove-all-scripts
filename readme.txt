Remove all enqueued scripts from every post/page/etc on your WordPress site.


== Description ==

Sometimes you just don't need all those scripts and you can't be 'arsed removing them individually.


== Installation ==

1. Download the plugin (zip file).
2. Upload and activate the plugin through the "Plugins" menu in the WordPress admin.
